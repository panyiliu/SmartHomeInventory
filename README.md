# 小仓屋（Flask）

一个简洁的本地 Web 小工具：把家里冰箱/橱柜里的食材、调料等录入进去，随时查看库存，并支持新增、编辑、增减数量、删除。

## 功能

- 库存列表（按更新时间排序）
- 搜索（名称/备注/单位）
- 按分类、位置筛选
- 新增食材
- 编辑食材
- 快速调整数量（±1、±0.5、自定义 delta）
- 删除（代表用完/不再需要）
- AI 图片识别录入（可选，需要配置环境变量）
- AI 文本解析录入（可选，需要配置环境变量）
- 扫码条码入库（可选，需要在“设置”里配置条码接口密钥）

数据存储在本机 SQLite：`instance/fridge.db`

## 运行（Windows / PowerShell）

在项目根目录执行：

```powershell
py -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
py .\app.py
```

启动后打开浏览器访问：

- `http://127.0.0.1:5000/`

## 可选配置（推荐用环境变量）

- `VOLCENGINE_API_KEY`: 启用火山引擎（Ark）AI 图片识别 / AI 文本解析（未设置时相关接口会报错，页面仍可手动录入）
- `FRIDGE_BARCODE_APP_SECRET`: 条码查询接口密钥（也可在“设置”页保存到本地数据库，但不建议）
- `FRIDGE_SMTP_PASSWORD`: SMTP 授权码/密码（也可在“设置”页保存到本地数据库，但不建议）
- `SECRET_KEY`: Flask 会话密钥（若要长期使用或部署，务必改为随机强密码）

## 说明

- 这是本地个人使用的基础版，没有做账号登录与权限控制。
- `SECRET_KEY` 目前是开发用固定值；如果你要部署到公网或多用户使用，请先改成随机强密码。
